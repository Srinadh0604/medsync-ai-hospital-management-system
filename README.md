# MedSync – AI-Powered Hospital Management System

## Overview

MedSync is a full-stack backend hospital management system built using Spring Boot.

The system manages patients, doctors, appointments, reports, authentication, and AI-powered patient summaries.

## Features

### Authentication & Security
- JWT Authentication
- Spring Security
- Role-Based Access Control (RBAC)

### Hospital Management
- Patient Management
- Doctor Management
- Appointment Scheduling

### AI Features
- Gemini AI Patient Summary Generation
- AI Health Recommendations

### Reporting
- PDF Report Generation
- Thymeleaf Templates
- AWS S3 Storage
- Email Delivery

### Monitoring
- Audit Logging
- Dashboard Analytics
- Health Check APIs

### DevOps
- Docker
- Docker Compose
